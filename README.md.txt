# College Website CMS

A responsive college website developed using HTML, CSS, Bootstrap, JavaScript, Spring Boot, and MySQL.

## Features

- Responsive UI
- Dynamic News
- Dynamic Notifications
- Google Sheet Integration
- Scheduled Data Synchronization
- REST APIs
- Academic Calendar
- Contact Page
- Kannada Language Support

## Tech Stack

Frontend
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

Backend
- Spring Boot
- Spring Data JPA
- MySQL

Tools
- Google Sheets
- Git
- GitHub

## Architecture

Google Sheet
↓
Spring Boot Scheduler
↓
MySQL
↓
REST APIs
↓
Responsive Website
